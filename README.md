# React Redux Calculator

## How to Run

### 1) Clone the repo

### 2) cd into `clinicsense`

### 3) `npm install` (only once)

### 4) `npm start`

## About the project

A React-Redux calculator without an equals (`=`) button. The result automatically updates as the user provides input
